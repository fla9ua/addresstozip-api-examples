# 郵便番号API サンプル集 (Address to ZIP)

[Address to ZIP](https://addresstozip.jp) は、日本の住所⇄郵便番号を変換する郵便番号APIです。
日本郵便の公式データに毎月追従し、登録から数分でAPIキーを発行できます(14日間無料、以降月額¥1,980)。

このリポジトリは、各言語からの最小の呼び出しサンプルをまとめたものです。
